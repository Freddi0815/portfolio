# Lebenslauf Website

This project is a modern resume website designed for showcasing personal information, skills, experience, and education, as well as providing a section for cover letters. The website is mobile-friendly and includes various interactive features to enhance user experience.

## Project Structure

```
lebenslauf
├── src
│   ├── css
│   │   ├── main.css          # Main styles for layout, typography, and color schemes
│   │   └── responsive.css     # Mobile-friendly styles and media queries
│   ├── js
│   │   ├── main.js           # Main JavaScript functionality
│   │   └── animations.js      # JavaScript for animations and transitions
│   └── assets
│       └── fonts             # Custom fonts for unique typography
├── index.html                # Main HTML file for the resume
├── anschreiben.html          # HTML file for cover letters
├── .gitignore                # Specifies files to be ignored by Git
└── README.md                 # Documentation for the project
```

## Features

- **Modern Design**: The website utilizes a clean and modern aesthetic with a focus on usability.
- **Mobile-Friendly**: Responsive design ensures the website looks great on all devices.
- **Interactive Elements**: Includes hover effects and animations to enhance user engagement.
- **Cover Letter Section**: A dedicated page for cover letters that can be linked from the main page.

## Setup Instructions

1. Clone the repository from GitHub:
   ```
   git clone https://github.com/Freddi0815/portfolio.git
   ```
2. Navigate to the project directory:
   ```
   cd portfolio/lebenslauf
   ```
3. Open `index.html` in your web browser to view the website.
